Etiqueas semanticas estructurales

div Es una etiqueta contenedora que no tiene semantica
header Cabecera de un sitio web o de una seccion
main Define la seccion principal de un documento (solo puede existir uno por doc)
footer Pie de pagina de un sitio web o de una seccion
nav Representa una navegacion
article Representa una seccion auto contenido (Se explica por si sola)
aside Contenido complementario o secundario (puede ser utilizado pra publicidad)
section Representa una seccion de contenido generico
address Representa informacion de contacto

Etiquetas de bloque vs etiquetas de linea

La etiqueta de bloque por exelencia es la div la etiqueta de bloque es aquella que genera saltos de linea con sus eqtiquetas hermanas.
La etiqueta de linea por exelencia es la span la etiqueta solo ocupa el espacio necesario que tiene su contenido. soy un span
